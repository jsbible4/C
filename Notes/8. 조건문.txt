1. if 조건문
    1) 특정 조건일 때 코드 실행 
    2) if 예
        - if 시계 알람이 울리면 
        {
            일어난다
        }

        if 날씨가 따듯하다면
        {
            코트를 얇게 입고 나간다
        }
    3) if (조건식)
       {
            코드        // 조건식이 만족할 때 실행할 코드 
       }
    4) if 문에서 중괄호 생략하기
        - 실행할 코드가 한 줄이라면 생략 가능 
        - 실행할 코드가 두 줄 이상일 때는 중괄호 생략에 주의 
            - 실행할 코드가 두 줄 이상이라면 반드시 중괄호로 묶어야 함 (안 그러면 한 줄만 인식함)

    5) if 조건문에서 정수외에 실수나 문자도 비교 가능함 

2. if 조건문 else 사용 (두방향으로 분기됨)
    1) 조건식이 만족할 때와 만족하지 않을 때 각각 다른 코드를 실행 
    2) 예
         if 물건 주문인가?
         {
            결제하고, SMS를 보낸다.
        
         }
         else
         {
            계속 쇼핑한다. 
         }
         
         if 네이버 사이트 회원인지? 그리고 여행동호회 회원인지?
         {
            글쓰기 화면 표시 
         }
         else
         {
            네이버 사이트 또는 여행동호회 회원이 아니므로 글을 쓸 수 없다는 팝업 문구 표시

         }
    3) else 는 if 조건문 뒤에 오며 단독으로 사용할 수 없음
    4) else 도 세미콜론을 붙이지 않도록 주의
    5) else 도 실행할 코드가 한 줄이면 중괄호 생략 가능
    6) if,  else로 실행할 코드가 두 줄 이상이면 반드시 중괄호로 묶어야 함
    7) 조건식을 여러 개 지정하기 
        - 논리 연산자를 사용하여 조건식을 여러 개 지정 가능
        - If 조건문은 안에 다시 if 조건문을 넣어서 여러 번 판단 가능

3. if 조건문 else if 사용
    1) else if 를 사용하여 여러 방향으로 분기 가능
        - 조건식을 여러 개 지정하여 각 조건마다 다른 코드를 실행가능 
        - else if는 단독으로 사용할 수 없음 
        - else if 다음에 조건식을 지정하지 않으면 컴파일 에러 발생
        - else if도 끝에 세미콜론을 붙이면 안됨
    2) 예
        if 치약 버튼을 눌렀다면 
        {
            치약을 내보냄
        }
        else if 칫솔 버튼을 눌렀다면
        {
            칫솔을 내보냄
        }
        else if 면도기 버튼을 눌렀다면
        {
            면도기를 내보냄
        }
        else 
        {
            아무 작동 되지 않음
        }
    
    3) if else if, else를 모두 사용하기
        - else if 앞에 else가 오면 컴파일 에러가 발생

4. 삼항 연산자 사용하기
    1) 변수 ? 값1 : 값2
    2) 예약어는
                참일 때 사용할 값
                ---
        - Num1 ? 100 : 200;
                      ----
                      거짓일 때 사용할 값
          ----
          참/거짓을 판단할 변수

        - num1 == 10 ? 100 : 200;
