1. 소프트웨어와 환경
    1) 일상 생활과 소프트웨어와의 융합
    2) 요즘은 사회 모든 분야에서 프로그래밍을 배움
    3) 문제 해결을 위한 과학적 사고를 기르는 것
        - 문제(서비스, 발주) 접근 방법
        - 복잡한 문제 => 작은 문제들

2. 프로그램
    1) ENIAC (최초의 컴퓨터)
    2) 폰 노이만에 의해 최초로 제안된 프로그램 내장 방식
    3) 프로그램을
        - 어떤 목적을 위해 진행 계획을 순서대로 표현 한 것
        - 비유) 문서로 완성된 조리 방법
    4) 프로그래밍
        - 컴퓨터(하드웨어)를 이용하여 문제를 해결하기 위해 작성한 명령들의 집합
        - 소프트웨어 개발 (코딩)
        - 프로그램을 작성해가는 과정

3. 프로그래밍 언어
    1) 사람이 컴퓨터와 의사소통을 하기 위한 수단
    2) 컴퓨터의 프로세서(cpu)는 자신만의 독자적인 기계어를 사용
    3) 언어 종류
        - 기계어, 어셈블리 언어, 고급 프로그래밍 언어
    4) 컴파일러(compiler) 
        - C 언어는 컴파일과정을 통해 기계어로 직접 번역되고 실행
        - 한 언어를 다른 언어로 변환하는 소프트웨어 
        - C 언어 => 컴파일러 => 기계어 (실행 파일) => 운영체제 => 실행 결과
